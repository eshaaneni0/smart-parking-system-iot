# Smart Parking System using IoT

An IoT-based smart parking system developed using ESP32, ultrasonic sensors, a servo motor, and MQTT communication. The system monitors parking slot availability, controls gate operations automatically, and provides real-time parking updates.

---

## Features

- Real-time parking slot monitoring
- Automatic entry and exit detection
- Servo motor gate control
- Vehicle counting system
- Parking full detection
- MQTT-based communication
- WiFi connectivity using ESP32

---
